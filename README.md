# Ragging Report

A comprehensive anti-ragging reporting system designed to help students report incidents of ragging safely and anonymously.

## Features

- **Anonymous Reporting**: Students can report ragging incidents without revealing their identity
- **Status Tracking**: Track the status of your reports in real-time
- **Admin Panel**: Administrators can manage and respond to reports
- **Image Upload**: Attach evidence images to reports
- **Rules & Guidelines**: Access to institutional anti-ragging policies

## Project Structure

- `ragging_report_login.html` - Login page for the reporting system
- `ragging_report_logic.js` - Core JavaScript logic for the application
- `stuhome.html` - Student home page after login
- `report.html` - Report submission page
- `track.html` - Track status of submitted reports
- `rules.html` - Anti-ragging rules and guidelines
- `about.html` - About the system
- `admin.htm` - Admin panel for managing reports

## Usage

1. Open `ragging_report_login.html` in a web browser
2. Login with your credentials or use anonymous reporting
3. Fill out the report form with incident details
4. Upload any relevant evidence (images)
5. Submit the report and use the tracking feature to monitor its status
